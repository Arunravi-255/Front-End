🚀 Overview

Programmatic access to all Prolio AI features. Build integrations, automate resume creation, generate portfolios and leverage AI at scale.

ℹ️
Base URL: https://api.prolioai.com/v2  ·  All requests must use HTTPS  ·  All responses are JSON

⚡ Quick Example

bash
# Create a resume with AI enhancement
curl -X POST https://api.prolioai.com/v2/resumes \
  -H "Authorization: Bearer sk-prolio-your_key" \
  -H "Content-Type: application/json" \
  -d '{"name":"Arun Ravi","role":"Full Stack Developer","ai_enhance":true}'

# Response
{
  "id": "res_9kXm3pQr",
  "status": "created",
  "url": "https://prolioai.com/r/res_9kXm3pQr"
}

⚡ Quickstart

From zero to first API call in under 5 minutes.

1
Get your API key
Copy your secret key from API Keys. Store it as PROLIO_API_KEY environment variable.
2
Install the SDK
bash
npm install @prolio/sdk    # Node.js
pip install prolio-ai      # Python
go get github.com/prolioai/go-sdk  # Go
3
Make your first call
javascript
const { ProlioAI } = require('@prolio/sdk');
const client = new ProlioAI({ apiKey: process.env.PROLIO_API_KEY });
// List resumes
const resumes = await client.resumes.list();
console.log(resumes.data.length);
// Enhance content with AI
const enhanced = await client.ai.enhance({ text: 'Worked on web apps', type: 'experience' });
console.log(enhanced.result);

📄 Resumes API

Create, retrieve, update, export and delete resumes programmatically.

GET/resumesList all resumes›

Returns a paginated list of all resumes for the authenticated user.

ParameterTypeRequiredDescription
limitintegeroptionalResults per page. Default 20, max 100.
statusstringoptionaldraft or published
json · 200 OK
{ "object":"list", "total":3, "has_more":false,
  "data": [{
    "id":"res_9kXm3pQr", "name":"Frontend Dev Resume",
    "status":"published", "template":"midnight",
    "url":"https://prolioai.com/r/res_9kXm3pQr"
  }]
}
POST/resumesCreate a resume›

Creates a new resume. Pass ai_enhance: true to auto-improve all descriptions using AI.

ParameterTypeRequiredDescription
namestringrequiredFull name of the owner.
rolestringrequiredJob title / professional role.
templatestringoptionalTemplate ID. Default minimal.
ai_enhancebooleanoptionalRun AI enhancement on all fields. Default false.
skillsarrayoptionalArray of skill strings.
experiencearrayoptionalArray of experience objects with role, company, start, end, desc.
GET/resumes/{id}/exportExport as PDF / DOCX›

Exports a resume as a file. Returns a signed download URL valid for 15 minutes.

ParameterTypeRequiredDescription
formatstringoptionalpdf | docx | json. Default pdf.
DELETE/resumes/{id}Delete a resume›

Permanently deletes a resume. This action cannot be undone. Returns 204 No Content on success.

🔐 Verify Signature (Node.js)

javascript
const crypto = require('crypto');
function verifyWebhook(payload, sig, secret) {
  const expected = crypto.createHmac('sha256', secret).update(payload).digest('hex');
  return crypto.timingSafeEqual(Buffer.from(expected), Buffer.from(sig));
}
